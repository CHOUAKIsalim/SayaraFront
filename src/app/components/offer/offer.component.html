Nos offres

Nous vous offrons plusieurs services que ce soit pour les fabricants ou les automoboliste

Gérez votre stock

Uploadez la liste des véhicules actuellement disponibles dans votre stock sous format csv et laissez notre application gérer votre stock.

Consultez vos commandes

Consultez les commandes en cours pour validation ou rejet. Si une commande est rejetée, le véhicule associé est immédiatement libéré. Si elle est validée, le véhicule associé n’est plus considéré disponible dans le stock.

Gérez les tarifs

Uploadez un fichier CSV contenant une liste de lignes de tarifs. Ce fichier sera utilisé pour le calcul des prix des véhicules pour les commandes.
